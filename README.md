# IT2311 Assignment – Task 1a: Data Preparation

**Done by:** Clifton Chen Yi, 231220B

This repository contains the Jupyter Notebook for Task 1a (Data Preparation) of the IT2311 Assignment, which performs data understanding and cleaning on the World Bank project documents dataset.

## How to Download the Notebook

### Option 1: Download a single file from GitHub

1. Navigate to the file **`231220B_DataPreparation.ipynb`** in this repository.
2. Click the **Download raw file** button (download icon) near the top-right of the file view.

### Option 2: Clone the entire repository

```bash
git clone https://github.com/Clif8893/cc.git
cd cc
```

The notebook file `231220B_DataPreparation.ipynb` will be in the root of the cloned directory.

### Option 3: Download as ZIP

1. Click the green **Code** button on the repository page.
2. Select **Download ZIP**.
3. Extract the ZIP file — the notebook will be inside.

## Running the Notebook

1. Make sure you have Python 3 and Jupyter installed:
   ```bash
   pip install jupyter pandas numpy matplotlib seaborn nltk
   ```
2. Place the dataset file `Task_1_TM_world_bank_projects_subset.json` in the same directory as the notebook.
3. Launch Jupyter:
   ```bash
   jupyter notebook 231220B_DataPreparation.ipynb
   ```
4. Run all cells in order.
